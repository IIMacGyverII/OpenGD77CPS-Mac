# OpenGD77 CPS — PriInterPhone fork v1.4.0

**Date:** June 5, 2026

## Tier 1.2 — Dark dock tabs

- MDI document tabs use the fork navy palette (`Theme.ApplyDarkDockPanelSkin`).
- `dockPanel.Skin` is now applied (skin was configured in v1.2.x but never assigned).

## Tier 2.8 — Channel grid bulk edit

Right-click selected channels in the grid:

- Set Power…
- Set Bandwidth…
- Set Squelch…
- Set Channel Mode… (digital channels only)
- Assign TG List… (Android CSV / Rx Group name)

## Tier 2.9 — Column visibility

Right-click a column header → toggle which columns are shown. Preference saved as `ChannelsGridHiddenColumns` in Setup profile.
