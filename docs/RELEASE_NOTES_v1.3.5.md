## Fix — custom ADB push folder names

Export & push no longer requires `YYYYMMDD_HHmmss`. Use any safe folder label (e.g. `June 5 2026`). Timestamp default still offered for new backups.

Invalid path characters (`\ / : * ? " < > |`) are still rejected.
