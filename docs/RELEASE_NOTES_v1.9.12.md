- **`OpenGD77CPS.exe --studio`** opens Codeplug Studio without showing the full CPS shell (no dock/tree/workspace flash)
- Main window stays hidden until you choose **Open full editor…**
- Studio gets taskbar icon + app icon on standalone launch
- Optional folder: `--studio "C:\path\to\backup"`
