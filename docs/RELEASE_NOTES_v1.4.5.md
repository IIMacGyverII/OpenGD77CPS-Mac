## Tier 3.4 — DMR ID lookup (RadioID.net)

Right-click a DMR ID and choose **Look up on RadioID.net** to open the registration page in your default browser.

- **Contacts grid** — right-click the Call ID column.
- **Contact editor** — right-click the Call ID field.
- **Channel editor** — right-click the TX Contact field (uses the selected contact's DMR ID).

All-call IDs (`16777215`) and invalid IDs do not show the menu item.
