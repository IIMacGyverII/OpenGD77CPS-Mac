## Health report click-to-open (F7 drill-down)

The F7 codeplug health report is now **interactive**:

- **Click** any channel, contact, or zone name in the HTML report to open that item in the editor.
- Covers relay=0, orphan TX contact, duplicate channel/contact names, duplicate DMR IDs, digital-no-contact, empty zones, channels not in any zone, and zone breakdown rows.
- Report stays open (non-modal) so you can fix several warnings in one session; press **F7** again to refresh counts.
